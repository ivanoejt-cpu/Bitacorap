⚙️ Configuración del Sistema

Tu llave se guarda localmente y no se comparte. Consíguela gratis en Google AI Studio.

Google Drive — Client ID OAuth 2.0
Pasos para obtener tu Client ID:
1. Ve a console.cloud.google.com
2. Crea un proyecto → Activa la Google Drive API
3. Ve a Credenciales → Crear → ID de cliente OAuth 2.0
4. Tipo: Aplicación web
5. En "Orígenes permitidos" agrega: file:// y la URL donde usas la app
6. Copia el Client ID y pégalo arriba
📂 Google Drive — File ID del Backup
¿Cómo obtener el File ID?
1. Abre tu Google Drive en el navegador
2. Busca el archivo Bitacorap_Backup.json
3. Haz clic derecho → Compartir o Obtener enlace
4. En la URL encontrarás algo como: /d/1A2B3C...xyz/view
5. Copia solo esa cadena larga y pégala arriba
💡 Solo necesitas hacer esto una vez en cada dispositivo nuevo.
